# ACSA SDF / WSP / ATR Reporting Concept

## The Problem

WSP and ATR data exists, but it is difficult to turn it into clean, repeatable management insight. The current risk is not only data capture, but also consistency, reviewability and visibility.

## The Demo

The demo shows a possible workflow:

- load workbook data
- stage and review records
- manage requested, planned and achieved training
- search provider/course information
- book training
- view employee profiles
- report requested vs planned vs achieved training
- copy an executive summary

## The Recommended Route

Start with Tier 2: Excel + Power BI Reporting.

This allows ACSA to prove the reporting logic before investing in a full app build.

## Proposal Options

1. Tier 1: Excel Control Pack
2. Tier 2: Excel + Power BI Reporting
3. Tier 3: Full Microsoft App Build
