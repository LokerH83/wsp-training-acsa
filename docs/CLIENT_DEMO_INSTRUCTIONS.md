# Client Demo Instructions

## Option A: Open The Hosted Demo

Open the GitHub Pages link provided by Hayden.

Expected format:

https://<github-username>.github.io/wsp-training-acsa/

## Option B: Open Locally

Open index.html from the repository folder, or run:

```bash
python -m http.server 8092
```

Then open:

```text
http://localhost:8092
```

## Demo Walkthrough

1. Start on Overview.
2. Confirm the five navigation items are visible:
   - Overview
   - Load Workbook
   - Manage Training
   - People Profiles
   - Reports
3. Review the dashboard values.
4. Open Load Workbook.
5. Click Load Sample Workbook.
6. Review detected columns and required field checklist.
7. Click Apply To Demo Staging.
8. Open Manage Training.
9. Search a provider or course.
10. Use a selected course in the training form.
11. Save a requested, planned or achieved record.
12. Book training from a planned intervention.
13. Open People Profiles and search for a demo employee.
14. Open Reports.
15. Filter by region, division, provider, race, sex/gender, age band or status.
16. Copy the executive summary.

## Safety Reminder

Use only synthetic/demo data.

Do not upload private ACSA workbooks or real employee data into the public demo.
