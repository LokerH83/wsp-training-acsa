# Implementation Checklist

## 1. Pre-Sales

- [ ] SkillSet: Confirm client problem and desired outcome.
- [ ] SkillSet: Share demo link and demo data policy.
- [ ] SkillSet: Confirm recommended tier.
- [ ] Client: Confirm stakeholders and decision makers.

## 2. Discovery

- [ ] SkillSet: Run SDF process discovery session.
- [ ] SkillSet: Collect current workbook structure.
- [ ] Client: Explain current WSP / ATR process.
- [ ] Client: Confirm reporting periods and fiscal calendar.

## 3. Data Review

- [ ] SkillSet: Map workbook fields to data dictionary.
- [ ] SkillSet: Identify missing IDs and matching risks.
- [ ] Client: Confirm demographic field rules.
- [ ] Client: Confirm provider/course master data.

## 4. Template Setup

- [ ] SkillSet: Create workbook templates.
- [ ] SkillSet: Add validation tabs and notes.
- [ ] Client: Test template with sample records.

## 5. Import Setup

- [ ] SkillSet: Define import rules.
- [ ] SkillSet: Define duplicate and match checks.
- [ ] Client: Confirm who approves imported rows.

## 6. Power Apps Setup

- [ ] SkillSet: Define production tables.
- [ ] SkillSet: Configure app screens/modules.
- [ ] Client: Confirm production environment.

## 7. Power BI Setup

- [ ] SkillSet: Create data model.
- [ ] SkillSet: Create dashboard pages.
- [ ] Client: Confirm KPI definitions and filters.

## 8. Security Setup

- [ ] SkillSet: Configure roles.
- [ ] SkillSet: Configure sensitive field access.
- [ ] Client: Confirm named users and access levels.

## 9. Testing

- [ ] SkillSet: Run functional testing.
- [ ] SkillSet: Run report reconciliation testing.
- [ ] Client: Perform user acceptance testing.

## 10. Training

- [ ] SkillSet: Train SDF admin users.
- [ ] SkillSet: Train report viewers.
- [ ] Client: Confirm internal support owner.

## 11. Go-Live

- [ ] SkillSet: Complete go-live checklist.
- [ ] Client: Approve live data load.
- [ ] Client: Confirm first reporting cycle.

## 12. Support

- [ ] SkillSet: Start agreed support model.
- [ ] SkillSet: Log issues and change requests.
- [ ] Client: Prioritise enhancements.
