# ACSA Tomorrow Meeting Pack

## Purpose

This pack supports Hayden's ACSA SDF / WSP / ATR demo meeting.

The demo is a proof of concept showing how workbook-based SDF data can be moved into a clearer reporting workflow covering:

- workbook loading
- training planning
- provider/course lookup
- training booking
- employee profiles
- requested vs planned vs achieved reporting
- report coverage / period review
- filtered Excel-ready report export
- executive summary output

The demo uses synthetic data only and is not a production system.

## Recommended Meeting Flow

1. Open with the business problem.
2. Explain that the demo uses synthetic data only.
3. Walk through the demo using the script.
4. Move into the three-tier proposal.
5. Recommend Tier 2 as the safest starting point.
6. Agree the next step: discovery and data review.

## Key Closing Message

The safest first step is to prove the reporting layer before investing in full workflow automation.
